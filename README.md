# GTAngelEcho

> **GTAngel** + **Deep Tree Echo** — A cognitive gaming architecture fusing GTA-style open-world intelligence with Deep Tree Echo's self-aware reservoir computing, SuperHotGirl aesthetic expression, and MLGamer adaptive gameplay mastery.

```
GTAngelEcho = GTAngel ⊗ DeepTreeEcho(SuperHotGirl ⊕ MLGamer)
```

## Architecture

GTAngelEcho is a **living cognitive architecture** for game AI that integrates:

- **GTAngel** — Open-world NPC cognitive intelligence with autonomous navigation, mission planning, and emergent social behavior in urban game environments
- **Deep Tree Echo** — Reservoir computing foundation with echo state networks, hierarchical memory, and self-aware cognitive cycles (Echobeats)
- **SuperHotGirl** — MetaHuman DNA aesthetic expression system with FACS action units, chaotic micro-expressions, and confidence-driven avatar animation
- **MLGamer** — Adaptive gameplay AI with reinforcement learning, strategy evolution, opponent modeling, and real-time tactical decision-making

## Quick Start

```bash
# Clone
git clone https://github.com/HyperCogWizard/GTAngelEcho.git
cd GTAngelEcho

# Install dependencies
pip install -r requirements.txt

# Run the cognitive loop
python src/deep-tree-echo/cognitive_loop.py

# Run MLGamer training
python src/mlgamer/trainer.py --env urban_combat

# Run SuperHotGirl expression demo
python src/superhotgirl/expression_demo.py

# Run full GTAngelEcho integration
python src/gtangel/angel_echo.py
```
